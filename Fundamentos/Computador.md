# Computador

## Definição
Máquina de processamento de dados: recebe dados de entrada, processa-os e os devolve ou armazena.

## Relações (SPO)
- Computador → processa → [[Dados Binários]]
- Computador → composto por → [[Hardware]] e [[Software]]
- Computador → derivado de → [[Modelo de Von Neumann]]
- Computador → executa → [[Programa Armazenado]]

## Funcionamento Geral
Do ponto de vista interno, qualquer operação — jogos, navegação na Internet, planilhas, vídeos, impressão, transferência de arquivos — resume-se ao mesmo ciclo:

> **Entrada → Processamento → Saída/Armazenamento**

Todos os dados são representados internamente como [[Dados Binários]] (sequências de 0s e 1s).
