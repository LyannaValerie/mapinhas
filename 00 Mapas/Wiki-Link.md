# Wiki-Link

Nota ponte criada pelo Codex para resolver referências internas ao conceito de wikilink no vault.

Um wiki-link é uma ligação interna em formato Obsidian, como `Nota` entre colchetes duplos, usada para conectar conceitos sem depender da posição hierárquica do arquivo.
