# Gerenciamento de Energia pelo Sistema Operacional

Conteúdo criado pelo Codex.

## Definição

**Gerenciamento de energia pelo sistema operacional** é o conjunto de políticas e mecanismos que reduzem consumo de energia mantendo desempenho aceitável.

## Áreas de atuação

- suspensão de dispositivos ociosos
- redução dinâmica de frequência e tensão
- desligamento de tela e disco
- gerenciamento térmico
- monitoramento de bateria
- interface entre drivers e política de energia

## Relações (SPO)

- Gerenciamento de Energia pelo Sistema Operacional -> controla -> dispositivos ociosos
- Gerenciamento de Energia pelo Sistema Operacional -> depende de -> [[Driver]]
- Gerenciamento de Energia pelo Sistema Operacional -> reduz -> consumo de energia
- Gerenciamento de Energia pelo Sistema Operacional -> negocia -> desempenho e autonomia
