# Software

## Definição
Parte **lógica** do [[Computador]]: sequência de números que representam dados ou instruções para seu funcionamento.

## Relações (SPO)
- Software → é parte de → [[Computador]]
- Software → contrasta com → [[Hardware]]
- Software → composto por → [[Dados Binários]] e instruções
- Software → viabilizado por → [[Modelo de Von Neumann]]

## Nota Linguística
Em inglês, "software" é substantivo incontável — não existe a forma "softwares". Em português, o uso pluralizado é aceito por incorporação ao idioma, mas o original inglês não admite plural.

## Relação com o Modelo de Von Neumann
O [[Modelo de Von Neumann]] foi o primeiro a permitir que a programação do [[Computador]] fosse feita por software (conceito de [[Programa Armazenado]]), em vez de reconfigurações físicas do [[Hardware]].
