# Circuito Integrado

## Definição
Conjunto de [[Transistor|transistores]], resistores e outros componentes eletrônicos fabricados **em um único substrato semicondutor** (tipicamente silício). Permite colocar dezenas a bilhões de componentes em uma área milimétrica.

Inventado independentemente por **Jack Kilby** (Texas Instruments) e **Robert Noyce** (Fairchild Semiconductor) em **1958**.

## Relações (SPO)
- Circuito Integrado → integra → múltiplos [[Transistor|transistores]] em único substrato
- Circuito Integrado → fabricado por → processo litográfico — ver [[Fabricação de Chips]]
- Circuito Integrado → implementa → [[Portas Lógicas]], memórias, [[Processador|processadores]]
- Circuito Integrado → quantidade de transistores segue → [[Lei de Moore]]
- Circuito Integrado → define → terceira geração de computadores (1965–1980)

## Impacto histórico
Antes do CI, cada [[Transistor]] era um componente discreto soldado manualmente. O CI reduziu drasticamente custo, tamanho e falhas — tornando viável o computador pessoal.

| Geração | Tecnologia | Período |
|---|---|---|
| 1ª | Válvulas termiônicas | 1945–1955 |
| 2ª | [[Transistor|Transistores]] discretos | 1955–1965 |
| **3ª** | **Circuitos Integrados** | **1965–1980** |
| 4ª | VLSI (Very Large Scale Integration) | 1980–presente |

## Escala de integração

| Sigla | Nome | Transistores por chip |
|---|---|---|
| SSI | Small-Scale Integration | < 100 |
| MSI | Medium-Scale Integration | 100 – 3.000 |
| LSI | Large-Scale Integration | 3.000 – 100.000 |
| VLSI | Very Large Scale Integration | > 100.000 |
| ULSI | Ultra Large Scale Integration | > 1.000.000 |

Processadores modernos (ex: Apple M-series, AMD Zen 4) contêm dezenas de **bilhões** de transistores.
