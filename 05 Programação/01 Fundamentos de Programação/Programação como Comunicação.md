# Programação como Comunicação

## Definição

Programação não é só instruir a máquina. Programa também comunica uma solução a pessoas que vão ler, modificar, depurar e manter o código depois.

## Relações (SPO)
- Programa → comunica → instruções ao computador
- Programa → comunica → intenção a programadores futuros
- Programador → resolve → problema
- Código legível → reduz → custo de manutenção
- Estilo → torna explícita → estrutura da solução

## Consequência prática

Programa deve funcionar e também expor sua lógica com clareza. Nomes, estrutura, decomposição e comentários úteis fazem parte da solução, porque manutenção é caso normal, não exceção.
